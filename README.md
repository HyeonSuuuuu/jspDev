## 커밋 컨벤션

| 타입     | 설명                                         |
| -------- | -------------------------------------------- |
| Feat     | 새로운 기능 추가                              |
| Fix      | 버그 수정                                    |
| Docs     | 문서 수정                                    |
| Style    | 코드 포맷팅, 스타일 변경 (비즈니스 로직 미변경) |
| Refactor | 코드 리팩터링                               |
| Test     | 테스트 추가/수정                            |
| Chore    | 빌드 업무, 기타 수정 (코드 기능이나 버그 수정과는 관련 없는 유지보수 작업) |
